# AI Chess Playground

1. In a terminal or command prompt, run `git clone https://github.com/ericguo202/ai-chess-playground.git`
2. Create a .env file in the project's root directory and supply API keys for OpenAI, Anthropic, Google, Grok, and Deepseek.
3. Install UV
4. In the command line, run `uv sync`
5. After all of the dependencies have been installed, run the app using `uv run app.py`
